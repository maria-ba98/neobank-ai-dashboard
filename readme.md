# 🏦 NeoBank AI Banking Dashboard

An AI-powered banking analytics dashboard built using Python, Streamlit, SQLite, and Machine Learning.

---

## 📌 Project Overview

This project simulates a modern European digital bank system with:

- Loans Analysis
- Deposits Analysis
- Net Profit Calculation
- Customer Segmentation
- Loan Default Prediction
- Banking KPIs
- Interactive Dashboard

---

## 🚀 Features

### 📊 Financial Analytics
- Total Loans
- Total Deposits
- Loan Profit Forecast
- Deposit Cost Analysis
- Net Profit Calculation

### 🤖 Machine Learning
- Loan Default Prediction
- Interactive Risk Prediction
- Random Forest Classifier

### 🧠 Customer Segmentation
- K-Means Clustering
- High / Medium / Low Value Customers
- Segment Profit Analysis

---

## 🛠 Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- NumPy
- Scikit-Learn

---

## ▶️ How to Run

Install dependencies:

pip install -r requirements.txt
Run the app:

streamlit run neobank.py
